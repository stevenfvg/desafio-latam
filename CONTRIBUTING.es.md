# Guía de contribución

## 🚫 No se aceptan contribuciones

¡Gracias por tu interés en este repositorio! Sin embargo, **este proyecto NO acepta contribuciones, pull requests ni modificaciones**.

### ¿Por qué?

Este repositorio funciona como un **archivo personal de aprendizaje** que contiene apuntes y ejemplos del Bootcamp de Desarrollo Full Stack en Desafío Latam. El contenido está pensado para:

- ✅ **Consulta y referencia**
- ✅ **Estudio educativo**
- ✅ **Inspiración para tus propios proyectos**

### Lo que SÍ puedes hacer

- 📖 **Leer y estudiar** el código
- 🔗 **Referenciar este repositorio** en tus propios materiales de aprendizaje (con la atribución correspondiente)
- 💬 **Abrir issues** para reportar errores o sugerir mejoras (solo con fines informativos)

### Lo que NO puedes hacer

- ❌ Enviar pull requests (serán cerrados)
- ❌ Hacer fork y modificar el código para redistribuirlo
- ❌ Usar este código con fines comerciales
- ❌ Crear obras derivadas

### Licencia

Este proyecto está licenciado bajo la [Licencia Internacional Creative Commons Attribution-NonCommercial-NoDerivatives 4.0 (CC BY-NC-ND 4.0)](LICENSE).

Esto significa que puedes ver y compartir el código con atribución, pero no puedes modificarlo ni usarlo con fines comerciales.

Este repositorio incluirá eventualmente un sitio web de documentación bilingüe para explorar proyectos, ejemplos y código fuente en inglés y español. Mientras tanto, consulta el [README.es.md](README.es.md) para obtener una visión general del contenido disponible.

## 📧 ¿Preguntas?

Si tienes preguntas sobre el código o el contenido, puedes:
- Abrir un issue para discutirlo
- Contactar al propietario del repositorio

**¡Gracias por comprender y respetar estas directrices!** 🙏
