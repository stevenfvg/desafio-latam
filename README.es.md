### 📋 Requisitos

Antes de ejecutar los ejemplos, asegúrate de tener Node.js instalado en tu sistema. Puedes verificar si Node.js está instalado ejecutando el siguiente comando en tu terminal o línea de comandos:

```bash
node --version
```

### 💻 Ejecutar los ejemplos

Para ejecutar los ejemplos debes iniciar la aplicación con Vite.js. Sigue estos pasos:

```bash
# ingresa al directorio basic-example
cd basic-example/

# Ejecuta el servidor de vite.js en modo desarrollo
npm run dev
```
