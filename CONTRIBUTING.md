# Contributing Guidelines

## 🚫 No Contributions Accepted

Thank you for your interest in this repository! However, **this project does NOT accept contributions, pull requests, or modifications**.

### Why?

This repository serves as a **personal learning archive** containing notes and examples from the Full Stack Development Bootcamp at Desafío Latam. The content is intended for:

- ✅ **Viewing and reference purposes**
- ✅ **Educational study**
- ✅ **Inspiration for your own projects**

### What You CAN Do

- 📖 **Read and study** the code
- 🔗 **Reference this repository** in your own learning materials (with proper attribution)
- 💬 **Open issues** to report bugs or suggest improvements (for information purposes only)

### What You CANNOT Do

- ❌ Submit pull requests (they will be closed)
- ❌ Fork and modify the code for redistribution
- ❌ Use this code commercially
- ❌ Create derivative works

If you have questions about the code or content, feel free to:
- Open an issue for discussion

**Thank you for understanding and respecting these guidelines!** 🙏
